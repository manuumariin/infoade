Detailed Description

T.D.A. Paises.

Una instancia c del tipo de datos abstracto Paises es un objeto que representa un conjunto de países, cada uno con sus datos geográficos y demográficos. Internamente se implementa como un conjunto ordenado de objetos de la clase Pais.
